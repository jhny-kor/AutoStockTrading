# 국장 체크리스트

## 실거래 전 체크리스트

아래 조건을 모두 확인하기 전에는 실거래를 켜지 않는 것을 권장합니다.

1. `KIS_USE_VIRTUAL=false`로 실전 서버 설정 여부 확인
2. `KIS_ALLOW_LIVE_ORDERS=true` 설정 여부 확인
3. `KR_LONG_DRY_RUN=false` 또는 `KR_SHORT_DRY_RUN=false` 전환 여부 확인
4. 계좌번호(`KIS_ACCOUNT_NO`)와 상품코드(`KIS_ACCOUNT_PRODUCT_CODE`) 확인
5. 장타/단타 예산(`ORDER_BUDGET_KRW`)이 의도한 수준인지 확인
6. 일일 주문 횟수 제한(`MAX_ORDERS_PER_DAY`) 확인
7. 최근 분석 로그가 오늘 날짜로 정상 생성되는지 확인
8. 최근 시그널 로그가 오늘 날짜로 정상 생성되는지 확인
9. 텔레그램 알림이 정상 동작하는지 확인
10. 중요한 공시 감시기가 실행 중인지 확인

## 장타 운용 체크리스트

- 장타 후보가 `1~3개` 수준인지 확인
- 동일 섹터 과집중 여부 확인
- 밸류업/자사주/배당 관련 공시가 있는지 확인
- 직전 급등 후 추격 진입이 아닌지 확인
- 후보 종목이 `20일선 위`, `60일선 위`인지 확인

## 단타 운용 체크리스트

- 거래량 배수가 너무 낮지 않은지 확인
- 당일 급락/급등 과열 구간인지 확인
- 중소형주는 뉴스/공시 변동성 확인
- 한 종목 집중 주문이 발생하지 않도록 제한 확인
- 단타 주문 엔진은 장타와 별도 예산으로 운용

## 매일 확인할 상태

- `collector` 실행 여부
- `kr_long` 실행 여부
- `kr_short` 실행 여부
- `reporter` 실행 여부
- `disclosure` 실행 여부
- 필요 시 `kr_long_trade`, `kr_short_trade` 실행 여부

## 비상시 체크리스트

- 시장 급락 시 주문 엔진 중지
- 중요 악재 공시 발생 시 해당 종목 전략 후보 제외
- 주문 응답 오류 시 live 설정 즉시 비활성화
- 중복 주문 발생 여부 로그 확인
